ComradeWallet 🇰🇪

A Full-Stack Expense Tracker for Students

ComradeWallet is a MERN stack application designed to help university students manage their finances. It allows users to track income, record expenses, and view a real-time summary of their financial health.
🚀 Live Demo

    Frontend: https://comrade-wallet-app.vercel.app

    Backend API: https://comrade-wallet-api.onrender.com

💡 About the Project

As a Business Information Technology (BIT) student at Dedan Kimathi University of Technology (DeKUT), I developed ComradeWallet to address the financial management challenges faced by university students. This project is more than just a coding exercise; it is a solution at the intersection of Finance and Technology.

By implementing secure JWT authentication and a cloud-based NoSQL database, I have demonstrated the ability to build robust, data-driven systems that prioritize user security and financial transparency—skills directly applicable to the digital banking and SACCO sectors.
✨ Features

    User Authentication: Secure Login and Registration using JWT (JSON Web Tokens).

    Real-time Dashboard: View current balance, total income, and total expenses.

    Transaction Management: Add, view, and delete financial records.

    Cloud Hosted: Fully deployed backend and frontend with a cloud database.

🛠️ Tech Stack

    Frontend: React.js, Axios, React Router

    Backend: Node.js, Express.js

    Database: MongoDB Atlas (Mongoose ODM)

    Deployment: Vercel (Frontend) & Render (Backend)

⚙️ Installation & Setup

If you want to run this project locally:

    Clone the repository:
    Bash

    git clone https://github.com/kevobinkou/comrade-wallet.git

    Environment Variables:
    Create a .env file in the root of the backend directory and add:
    Code snippet

    MONGO_URI=your_mongodb_connection_string
    JWT_SECRET=your_secret_key
    PORT=5000

    Install Dependencies:
    Run npm install in both the frontend and backend folders.
